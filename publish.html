How do I publish my open source project?

Thanks for your interest in sharing your code! If you are a developer employed by NASA or a NASA contractor and have open source code you wish to host on the NASA GitHub repository, email us at opensource@lists.nasa.gov and follow the steps below. If you aren’t affiliated with NASA, we encourage your to clone, fork, modify, break, and improve the code that we’ve released thus far. We’re still working out the proper methods to take user contributions to our code and will provide information on how to do so as it develops.

NASA Officials: Beth Beck and Jason Duley
